## Background
- 산업·임상 연구는 자동화를 수용했으나, 학술 실험실은 여전히 대학원생·포닥의 저비용 노동에 의존
- 실험실은 야간·주말에 유휴 상태이며, 인력 이직으로 지식·경험 손실 발생
- AI와 자동화를 결합한 self-driving lab이 생물학, 화학, 재료과학 분야에서 반복 작업 설계·실행, 데이터 분석, 다음 실험 튜닝에 활용
- Nobel Turing Challenge는 30년 내 자율 실험실이 노벨상급 발견을 이룰 것이라 전망

## Key Experiment Methods
1. **Self-driving lab 정의**: 자동화 실험 장비 + AI/ML 통합, 데이터 분석 후 다음 실험 조건을 실시간으로 결정 - 고속실험(high-throughput)과 차별점은 실시간 데이터 처리와 실험 계획 피드백 루프
2. **Ada 시스템** (UBC Berlinguette lab): 로봇 암으로 perovskite 태양전지 박막 재료 제작, 최적화 파라미터: 첨가제 농도, 어닐링 시간, 어닐링 온도, 스핀 코팅 속도, 용매 조성 - 두께·형태·결함 화학이 공정 조건에 민감
3. **Cloud lab**: Carnegie Mellon University 학술 클라우드랩, Emerald Cloud Lab 원격 실험 주문 서비스 - 로봇 기기와 훈련된 기술자가 실험 수행
4. **자동화 워크플로우 연결의 어려움**: 상용 자동화 장비(GC autosampler, NMR autosampler 등) 간 연계가 기술적 장벽, Berlinguette "로봇이 실험 진화에 따라 진화해야 함"
5. **데이터 품질**: ML은 고품질·주석이 달린 데이터 스트림 필요, 로봇 자동화가 이를 제공 - "Machine learning really requires automated workflows"

## Results
- Self-driving lab 적용 사례: 스마트 재료 설계(Berlinguette Ada), 효소 발견(SAMPLE, Kanda LabDroid), 신물질 합성(A-Lab), 광촉매 최적화
- Ada 시스템: perovskite 박막 성능 최적화 자동 수행, 공정 조건 파라미터 공간에서 효율적 탐색
- Cloud lab: 연구자들이 온라인에서 실험 주문, 물리적 실험실 접근 없이 원격 연구 가능
- Nobel Turing Challenge: 2020년 컴퓨터 과학자들 시작, 30년 내 자율 실험실의 노벨상급 발견 목표
- Missy Cummings(George Mason Univ) 경고: "AI는 인간이 지원하는 도구, 인간 감독 필요"

## Perspective
- AI와 자동화의 결합이 기초 과학 효율성을 획기적으로 개선할 잠재력
- critics는 과장과 AI 시스템의 실패 가능성 경고 - 인간 감독 필요
- 자동화 인프라 구축이 선행되어야 AI/ML의 과학적 활용 가능
- Cloud lab이 실험실 접근성 민주화에 기여할 수 있음
- 자율 실험실의 확산은 과학 연구 패러다임 자체를 변화시킬 것
