## Background
- hiPSC 유래 세포는 개인 맞춤형 질병 모델링·약물 스크리닝에 유용하나, 배양·분화 프로토콜의 기술적 복잡성과 표준화 부재로 대규모 연구에 한계
- 부착 배양 기반 hiPSC 문화는 반복적 액체 처리(배지 흡인·분주, 세포 현탁)에 의존하여 자동화에 적합
- TECAN Fluent와 같은 상용 로봇 워크스테이션으로 재현 가능·확장 가능한 hiPSC 배양·분화 파이프라인 구축 필요
- AMD(황반변성) 환자 유래 hiPSC-RPE를 개인 맞춤형 약물 스크리닝에 활용

## Key Experiment Methods
1. **TECAN Fluent 780 워크스테이션**: 8-tip Air FCA(액체 처리), 96-well MCA, RGA(그리퍼 암), Liconic incubator(STX44), Hettich 원심분리기, Cytation 1 high-content imager 통합
2. **자동화 hiPSC 배양**: 6-well 부착 배양, daily media exchange 자동화 - plate 이송 → Cytation 1 이미징(confluence 측정) → tilt carrier(-5°) → FCA로 1000 µL tip으로 배지 흡인/분주 → incubator 복귀
3. **자동화 passaging**: hypertonic citrate solution 처리 6분(37°C) → wide bore tip으로 세포 탈착(12회 흡인/분주 패턴) → 800g 3분 원심 → pellet 재현탁(6 mL 단계적 추가, wide bore tip으로 3회 혼합)
4. **자동화 iPSC-RPE 분화**: hiPSC를 RPE로 분화, fully defined media 사용, 분화 인자(FGF receptor inhibitor, TGF-β/Activin/Nodal inhibitor, Wnt inhibitor 등) 자동 투여
5. **자동화 약물 테스트**: AMD 환자 유래 hiPSC-RPE에 mitoactive compound 노출 후 미토콘드리아 기능 자동 측정
6. **최적화 파라미터**: 액체 처리 볼륨, 흡인/분주 속도, tip 종류(standard/wide bore), tilt 각도, incubation 시간, 원심 조건, confluence 기반 passaging 타이밍

## Results
- TECAN Fluent로 여러 hiPSC 라인 배양·분화 성공, 수동 배양 대비 재현성·확장성 향상
- AMD 환자 유래 hiPSC-RPE 라인 자동 분화 성공, 하류 약물 테스트에 활용
- 자동화 약물 테스트 워크플로우로 mitoactive compound 노출 후 미토콘드리아 기능 측정
- 상용 플랫폼 기반 proof-of-principle로 방법론의 광범위 확산 가능
- hiPSC 배양·분화의 end-to-end 자동화가 개인 맞춤형 약물 스크리닝에 유효함 입증

## Perspective
- 상용 로봇 워크스테이션이 hiPSC 기반 연구·치료 개발 접근성 향상
- 자동화 이미징과 결합하여 장기간 문화 평가·데이터 획득 가능
- 환자 코호트 규모로 iPSC-RPE 생성 가능해져 개인 맞춤형 약물 스크리닝 실현
- GLP/cGMP 준수 자동화 파이프라인이 세포치료제 제조로 확장 가능
- 액체 처리 자동화가 hiPSC 연구의 표준화·재현성 개선에 핵심
