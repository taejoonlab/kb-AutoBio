# In-process evaluation of culture errors using morphology-based image analysis

## Background
- 세포치료제 산업적 규모 제조에서 품질 관리(QC)가 주요 병목 현상
- 효과적인 공정 중 측정 기술 부재로 시간 소모적·복잡한 세포배양 공정 품질 평가가 수동 현미경 관찰에 의존
- 환자 유래 MSC는 로트 간 변이가 크며, 제조 중 예기치 않은 품질 변화 유발 가능
- 형태학적 정보는 비침습적·정량적 공정 중 측정 도구로 잠재력 있음

## Key Experiment Methods
1. **의도적 오류 모델**: 인간 MSC(골수 유래 11로트, 지방 유래 10로트) 총 21개 로트를 표준 조건 + 4가지 의도적 오류로 배양: (a) 시약 농도 오류, (b) 배지 교환 누락, (c) 배양 온도 이탈, (d) 기계적 진동 오류
2. **시간 경과 형태학 프로파일링**: 위상차 현미경 이미지(2시간 간격)에서 세포 형태학적 특성 정량 측정 - 세포 면적, 원형도, 장축/단축 비율, 텍스처 특징량 등 다수 파라미터 추출
3. **수정 PCA 시각화**: 형태학 전이 선 플롯(morphological transition line plot)으로 로트 간·배양 조건 간 차이를 2D 공간에서 실시간 시각화, 표준 조건에서의 전이 궤적과 이탈 비교
4. **예측 모델링**: 선형 회귀 분석으로 세포 증식률 예측 및 오류 조건 판별, 2일 배양 데이터만으로 >80% 정확도 달성
5. **MT 방법(Mahalanobis-Taguchi)**: 표준 데이터는 풍부하지만 실패 데이터가 적은 제조 공정에 적합한 이상 탐지 머신러닝, 마할라노비스 거리로 표준 상태からの 이탈도 정량화

## Results
- 수정 PCA 시각화로 세포 로트 간 고유 형태학 궤적 확인, 표준 조건과 4가지 오류 조건이 명확히 분리 - 실시간 이상 탐지 가능
- 세포 증식률 예측 정확도 >80%, 오류 조건 판별 정확도 >80% -仅 2일 배양 데이터로 가능(전체 배양 기간 대비 초기 단계)
- MT 방법으로 표준 데이터 중심 제조 공정에서 이상 탐지 가능, 실패 데이터가 제한적인 실제 제조 환경에 적합
- 골수 유래 MSC와 지방 유래 MSC가 형태학 전이 궤적에서 명확히 구분, 로트 간 변이가 배양 조건 영향보다 큼
- 형태학 정보가 세포 제조 공정에서 효과적인 공정 중 측정 도구임 입증, 수동 현미경 관찰 대체 가능

## Perspective
- 비침습적 형태학 기반 이미지 분석이 세포치료제 제조 품질 관리의 핵심 도구
- 공정 중 실시간 모니터링으로 배양 오류 조기 발견 및 대응 가능
- MT 방법은 실패 데이터가 제한적인 실제 제조 환경에 적합
- 향후 자동화 이미징과 결합하여 세포 제조 파이프라인 전반에 적용 가능
- 규제 과학 측면에서 공정 중 측정 데이터가 제품 품질 보증에 기여
