# Automated microfluidic cell culture of stem cell derived dopaminergic neurons

## Background
- 파킨슨병은 중뇌 도파민 뉴런의 기능 장애·사망 특징, 환자 유래 in vitro 질병 모델 개발이 주요 과제
- 3D 미세유체 세포배양 장치에서 세포 장기 유지·모니터링 자동화 플랫폼 필요
- 수동 배양은 액체 처리(배지 분주·흡인)가 필수, 미세유체 배양은 저볼륨 처리 필요
- SiLA(Standardisation in Laboratory Automation) 표준 프로토콜로 이종 제조사 장치 통합
- Organ-on-a-Chip 자동화 배양과 time-lapse 이미징·전기생리학 모니터링 결합 필요

## Key Experiment Methods
1. **자동화 세포배양 플랫폼**: gantry robot(Cartesian 좌표 이동) + liquid handler(contact/non-contact dispensing) + incubator + microscope 통합, SiLA 표준 프로토콜로 통신 통합
2. **미세유체 Organ-on-a-Chip**: 3D 미세유체 장치에서 파킨슨병 환자 유래 neuroepithelial stem cell 배양, 중뇌 특이적 도파민 뉴런으로 분화
3. **Time-lapse 이미징 품질 관리**: 장기간 자동 이미징으로 세포 상태 모니터링, 분화 과정 추적
4. **전기생리학 모니터링**: calcium imaging으로 분화된 뉴런의 전기생리적 활동 확인, immunostaining으로 분화 효율 검증
5. **유연한 프로토콜 적응**: 다양한 실험 프로토콜로 유연하게 적응, 관류율(perfusion rate) 등 핵심 실험 파라미터 정량 제어
6. **최적화 파라미터**: dispensing speed/nanoliter volume, perfusion rate, cell culture conditions, fluid temperature, imaging interval, SiLA device class 설정

## Results
- 자동화 Organ-on-a-Chip 배양 플랫폼 최초 구현, 파킨슨병 환자 유래 세포 장기 배양 성공
- Time-lapse 이미징으로 도파민 뉴런 분화 과정 연속 모니터링
- Calcium imaging으로 분화된 뉴런의 전기생리적 활동 확인
- Immunostaining으로 분화 프로토콜 효율 검증
- SiLA 표준 프로토콜로 이종 제조사 장치 통합 성공, 단일 자율 플랜트로 기능
- 미세유체 배양에서 저볼륨 dispensing·aspiration 정확도 확보

## Perspective
- 자동화 Organ-on-a-Chip이 환자 맞춤형 질병 모델링에 기여
- SiLA 표준화가 실험실 자동화 장치 통합 장벽 해소
- Long-term 자동 배양이 신경퇴행성 질환 진행 연구에 유효
- 향후 다중 Organ-on-a-Chip 병렬 처리, 고처리량 약물 스크리닝으로 확장
- 자동화 이미징·전기생리학 모니터링 결합이 in vitro 질병 모델 검증 표준으로 발전
