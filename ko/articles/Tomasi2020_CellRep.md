# Individual Control and Quantification of 3D Spheroids in a High-Density Microfluidic Droplet Array

## Background
- 3D 세포배양(spheroid, organoid)이 확산됨에 따라 개별 세포 수준의 정량적 데이터 수집 도구 필요
- 기존 3D 배양은 벌크 방식으로 개별 스페로이드의 이질성 분석이 어려움
- 미세유체 칩 기반 드롭렛 기술은 개별 세포/스페로이드 포획 및 환경 제어 가능
- 단일 칩에서 다양한 조건(공배양, 하이드로젤 캡슐화, 약물 테스트) 병렬 적용 필요

## Key Experiment Methods
1. **고밀도 미세유체 드롭렛 어레이**: 모세관 앵커(capillary anchors)에 드롭렛 쌍을 순차적으로 포획, 표면 장력으로 위치 고정 후 전기장/유량 제어로 병합(merging)하여 환경 변조
2. **1 스페로이드/드롭렛**: 개별 스페로이드를 독립적으로 제어, 미세환경 변조를 드롭렛 병합으로 유도 - 공배양, 하이드로젤 캡슐화, 약물投與 모두 동일 플랫폼에서 구현
3. **약물 농도 범위 테스트**: 단일 칩에서 간유사 스페로이드에 광범위한 약물 농도 병렬 적용, 농도별 드롭렛을 다른 스페로이드 드롭렛과 병합
4. **데이터 기반 접근법**: 시간 경과 이미지 획득(분당 프레임) + 미세유체 결합으로 3D 조직 수준의 동적 약물 반응 시간 분해능 분석
5. **단일 세포 측정과 집단 측정 연계**: 스케일 간 이질성 분석, 개별 세포 추적과 전체 스페로이드 응답 비교

## Results
- 드롭렛 병합을 통해 공배양(서로 다른 세포 유형 초기화), 하이드로젤 캡슐화(3D 미세환경 구축), 길항적 상호작용 관찰 등 복잡한 프로토콜 단일 칩에서 구현
- 단일 칩에서 광범위한 약물 농도를 간유사 스페로이드에 병렬 적용, 농도-반응 곡선 한 실험으로 획득
- 시간 분해능 단일 세포 데이터로 스페로이드 수준에서 조절되는 동적 약물 반응 규명 - 개별 세포 응답 이질성이 스페로이드 전체 반응에 미치는 영향 정량화
- 데이터 기반 접근법으로 3D 배양 행동의 이질성을 스케일 간에 분석, 단일 세포 측정과 집단 측정 간 상관관계 규명
- 미세유체 드롭렛 플랫폼이 organoid/spheroid 기반 고속 약물 스크리닝에 유효함 입증

## Perspective
- 미세유체와 3D 세포배양의 융합이 organ-on-a-chip 플랫폼 발전에 기여
- 개별 스페로이드 제어를 통한 고속 약물 스크리닝 및 독성 평가 가능
- 데이터 기반 분석이 3D 배양 이질성 이해의 핵심
- 향후 다중 스페로이드 상호작용, 조직 공학, 개인 맞춤형 약물 테스트로 확장
- 자동화 이미징과 미세유체 제어 결합이 고속 3D 배양 분석의 방향성 제시
